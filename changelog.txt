list of the changelog published in the commits of GitHub

version V2.0 refers to the current development version of TRH mod.
alpha 1-02 refers to the alpha state of the version development; "-02" refers to the second iteration of the .pk3 file directory.
#### or any other number refers to the commit uploaded to GitHub; for example, 0001 is the initial upload and 0002 is the latest addition to the project.

and now the changelog:

#0001 (14/2/2026)
Initial upload.
Added in Alpha 1-02 (v00001):
    Weapon Base: All ammo classes replaced; ammo available in different quantities; implemented internal weapon magazine logic.
    Players: Added Cyborg_Rodri_Player and all Grappling Hook logic.
    Weapon Placeholders: Beta versions of weapons (pre-Alpha 2 logic). Includes: Pump_Shotgun, Super_Shotgun, Golden_Super_Shotgun, Machinegun, Rifle (all 3 variants), RPG_69, CV, and Traffic_Cone.
    Enemies: Base code for the Obliterated faction and the Obliterated_Shotgun actor.
    FX: Code for basic effects such as shell casings and bullet sparks (Bullet_Spark).

#0002 (14/2/2026)
Documentation: Obliterateds.
Added documentation for the Obliterated class in enemies.txt; written with assistance from Gemini AI. 

#0003 (15/2/2026)
Added Obliterated Rifle sprites.
Added 100 images (192x96px) for the new enemy's sprite set. 

#0004 (15/2/2026)
Added Obliterated drop logic and Obliterated Rifle AI. 

#0005 (15/2/2026)
Added weapons and logic:
    * Dual Shotgun sprites added.
    * Gatling sprites added.
    * Dual Shotgun logic: Alternating fire rate between left and right barrels.
    * Gatling logic: Dual-state machine with spin-up, spin-down, and AltFire toggle for instant firing.
    * Weapon class organization: Reordered source code to group all shotgun variants (Pump, SSG, Golden, Dual) before heavy weaponry.

#0006 (17/2/2026)
Armor System Update:
    * Added Armor Bonus sprite and logic.
    * GreenArmor replaced with Grey_Armor; BlueArmor replaced with Black_Armor.
    * Updated armor sprites.
    * Modified rocket spawns: 50/50 chance to spawn Rocket_Ammo_Box or Missile_Ammo_Box.

#0007 (17/2/2026)
Health System Update:
    * Added Health Bonus, Stimpack, and Medikit replacements to support the 400HP cap.
